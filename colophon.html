colophon

this site is owned, designed, edited, built, and maintained by nicole j. kim

this corner is a product of constant iteration, often with tweaks made at a minute scale. it is not meant to represent a polished, premeditated, "well-designed" system; rather, this is principally a site of spontaneous craft and sculpting, and the structure of its components (or lack thereof) will reflect this approach. for this reason, this site is created intentionally with code in a manner that would offer me control over its parts.

some additional details:

  • haphazardly pieced together using html5, css3, and javascript
  • source code hosted on github
  • typefaces used include space grotesk and dm sans
